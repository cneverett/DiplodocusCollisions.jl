# DiplodocusCollisions.jl

[![Build Status](https://github.com/cneverett/DiplodocusCollisions.jl/actions/workflows/CI.yml/badge.svg?branch=master)](https://github.com/cneverett/DiplodocusCollisions.jl/actions/workflows/CI.yml?query=branch%3Amaster)

*DiplodocusCollisions.jl* is a sub-package of the *Diplodocus.jl*, responsible for generating collision arrays for discrete particle iterations.

For documentation, tutorials and installation https://cneverett.github.io/Diplodocus.jl/
